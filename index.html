Sooksma Scramble

Instructions

Sooksma Scramble is a combination of Boggle and Scrabble

- You will have 16 tiles of randomly generated letters on a 4x4 board

- Letters each have a point value

- You will make as many words as possible in 60 seconds, per round, for 3 rounds

  • You must hit the points goal in order to advance to the next round
  • The point requirement will increase per round
  • Beat 3 rounds to win the game

- Words can be created using any letter on the board

  • Letters can only be used once per word
  • You can only use a word once per round for points

- Words must be at least 3 letters long

- If you don't like the letters you've been given, you can reset the board

  • Doing this will bring you back to round 1, with 0 points (yep - you have to start ALL over)
